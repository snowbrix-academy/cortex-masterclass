User Classification: Human vs Service Account

  • Human Users: Email-pattern usernames (jane.doe@company.com)
  • Service Accounts: Naming convention (SVC_*, SA_*, uppercase patterns)
  • System Queries: USER_NAME = 'SNOWFLAKE' (exclude from chargeback)

SP_CLASSIFY_BI_CHANNELS: Procedure classifies all queries and calculates cost by tool

Warehouse Isolation Strategy

Recommended Patterns:

  • Dedicated warehouses: WH_POWERBI_PROD, WH_TABLEAU_PROD (clear attribution)
  • Shared with QUERY_TAG: Enforce tag-based attribution for mixed tools
  • Anti-pattern: BI tools sharing ETL warehouses (attribution impossible)
